# Verificador-de-CPF
A Python application that validates Brazilian CPF numbers using the official verification digit algorithm.

## Features
- CPF format validation
- Verification digit calculation
- Input sanitization
- Clear user feedback
- Structured and maintainable code

## Technologies=
- Python 3

## Purpose
This project was developed to practice algorithm implementation, data validation, and clean code principles using a real-world Brazilian identification system.
